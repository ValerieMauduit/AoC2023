# AoC2023
Advent of Code 2023

To run the code:

```
python aoc2023.py --dir dir --day D --star S
```
with:

- dir is the input data directory name
- D is the day number
- S is the star number (1 or 2)

To run the tests:

```commandline
python test_dayXX.py 
```

in the right directory
